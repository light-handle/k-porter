K-Porter

A Modern Kubernetes Port Forwarding Utility

K-Porter is a desktop application that simplifies Kubernetes port forwarding. With an intuitive interface, it lets you browse your clusters, namespaces, and services, and set up port forwards with just a few clicks - no more complex kubectl commands.

How It Works

1

Install K-Porter

Download and install K-Porter for your operating system from the GitHub releases page.

2

Select a Cluster

K-Porter automatically detects your Kubernetes clusters from your kubeconfig file.

3

Browse & Connect

Browse namespaces and services, then click "Port Forward" to create a connection.

Available for All Platforms

K-Porter is available for macOS, Windows, and Linux. Installation is simple - just download from GitHub Releases and follow the platform-specific instructions.
