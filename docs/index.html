K-Porter

A simple, elegant Kubernetes port forwarding utility

K-Porter is a desktop application that simplifies Kubernetes port forwarding. It provides an intuitive interface to browse your clusters, namespaces, and services, allowing you to set up port forwards with just a few clicks.

No more typing complex kubectl commands or remembering port numbers - K-Porter handles it all for you.

Key Features

  • Automatic Detection - Automatically detects Kubernetes clusters from your kubeconfig file
  • Resource Browser - Easily browse namespaces and services in each cluster
  • One-Click Port Forwarding - Set up port forwarding with just a few clicks
  • Connection Manager - Track and manage all active port forwards
  • Multi-Cluster Support - Easily switch between different Kubernetes clusters
  • Dark Mode Support - Comfortable viewing experience day or night

Supported Platforms

K-Porter is available for:

  • macOS (10.14 and later)
  • Windows (10 and later)
  • Linux (Debian/Ubuntu and other distros)
