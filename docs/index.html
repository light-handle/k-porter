K-Porter

A Kubernetes Port Forwarding Utility

K-Porter is a desktop application that simplifies Kubernetes port forwarding. Browse your clusters, namespaces, and services, and set up port forwards with a simple interface instead of complex kubectl commands.

Download Latest Release | View on GitHub

Features:

  • Automatic detection of Kubernetes clusters from your kubeconfig file
  • Easy browsing of namespaces and services
  • One-click port forwarding to any service
  • Manage all active port forwards from a simple dashboard
  • Multi-cluster support
  • Available for macOS, Windows, and Linux
