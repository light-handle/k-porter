K-Porter

A Modern Kubernetes Port Forwarding Utility

Simplify your Kubernetes workflow with K-Porter - a sleek, intuitive desktop application that makes port-forwarding to Kubernetes services effortless. Browse clusters, namespaces, and services with ease, and set up port forwards with just a few clicks.

Download Now View on GitHub

🔍 Automatic Detection

K-Porter automatically detects all your Kubernetes clusters from your kubeconfig file, providing a seamless setup experience without any manual configuration.

📋 Resource Browser

Easily browse through all namespaces and services in each of your clusters with a clean, intuitive interface designed for maximum productivity.

🔌 One-Click Port Forwarding

Set up port forwarding to any service with just a few clicks. No more typing long kubectl commands or remembering port numbers.

📊 Connection Manager

Manage and monitor all your active port forwards from a single dashboard, with the ability to start, stop, or modify connections instantly.

🔄 Multi-Cluster Support

Effortlessly switch between different Kubernetes clusters, making it easy to manage multiple environments from a single application.

🌓 Dark Mode Support

K-Porter adapts to your system preferences with full light and dark mode support, providing a comfortable experience day or night.

How It Works

1

Install K-Porter

Download and install K-Porter for your operating system - available for macOS, Windows, and Linux.

2

Select a Cluster

K-Porter automatically detects your Kubernetes clusters from your kubeconfig file.

3

Browse & Connect

Browse namespaces and services, then click "Port Forward" to create a connection.
